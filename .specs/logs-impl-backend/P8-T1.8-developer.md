## Decisions

### Factory uses `require()` (not dynamic import)
`PgNotifyEventBus` creates a PG connection in its constructor, which must never happen in
test or Electron environments. The factory uses `require()` with a runtime guard
(`NODE_ENV === 'test'`). This keeps test startup fast and avoids connection errors in CI.

### PgNotifyEventBus — large event handling
pg_notify has a hard 8000-byte payload limit. Large events (completed, human_gate with full
data) would exceed it. Decision: store the full payload in `EventPayload` (already in schema)
and send a lightweight `event_ref` over the wire. The SSE route reads `EventPayload` for the
reconnect buffer anyway, so this is consistent.

### PgNotifyEventBus — LISTEN/UNLISTEN lifecycle
`subscribe()` ref-counts subscribers per project and issues `LISTEN` on the first and
`UNLISTEN` on the last. This prevents accumulating idle LISTEN channels on the PG connection
when no UI client is connected to a project stream.

### SSE routes — permission filtering
Events are not all equal: `cost_update`/`budget_warning` require `stream:costs`, and
`human_gate` requires `stream:gates`. The filter is applied inside the stream before
`controller.enqueue()`. A caller missing those permissions receives all other events but
never sees cost or gate data — partial visibility handled transparently.

### SSE routes — reconnect buffer
`Last-Event-ID` triggers a query on `EventPayload` for rows within the 24h TTL window.
Events are replayed in `created_at` order before live subscription begins. The replay is
non-blocking (async, fire-and-forget on error) to avoid delaying the first byte of the SSE
response.

### API key auth in SSE routes
Better Auth session lookup via `auth.api.getSession()`. API key fallback: hashed with SHA-256
and matched against `ProjectApiKey.key_hash`. `require('crypto')` used inline to avoid
top-level imports in Next.js edge-compatible routes. Note: API key auth in SSE routes is
a v1 convenience; a proper API key middleware will be extracted as part of T1.9/T2.x.
